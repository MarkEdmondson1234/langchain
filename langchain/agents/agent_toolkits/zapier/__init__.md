# Summary
This code defines a `ZapierToolkit` class that contains a list of `BaseTool` objects and two methods: `from_zapier_nla_wrapper` and `get_tools`. The `from_zapier_nla_wrapper` method takes in a `ZapierNLAWrapper` object and returns a `ZapierToolkit` object with `ZapierNLARunAction` tools that correspond to the actions available in the `ZapierNLAWrapper`. The `get_tools` method simply returns the list of tools in the toolkit.
